# 📄 Resume Builder (React + PDF Download)

A modern and user-friendly Resume Builder web application built using **React**. This project allows users to create, preview, and download professional resumes in real-time with multiple templates.

---

## 🚀 Features

### 🧑‍💼 1. User Information Input

* Personal Details (Name, Email, Phone, Address)
* Education Details
* Work Experience
* Skills
* Projects

---

### 👀 2. Live Resume Preview

* Real-time preview while typing
* Instant updates without page reload
* Structured and professional layout

---

### 🎨 3. Multiple Templates

* Choose from multiple resume designs
* Switch templates dynamically
* Clean and modern UI styles

---

### 📥 4. Download as PDF

* Export resume as a high-quality PDF
* Supports multi-page resumes
* Maintains layout consistency

---

### 🔄 5. Dynamic Sections

* Add/remove education, experience, and projects
* Flexible form handling
* User-friendly controls

---

### 💾 6. Local Storage Support

* Automatically saves user data in browser
* Data persists even after refresh

---

### 🌙 7. Dark Mode (Optional)

* Toggle between light and dark themes
* Improved user experience

---

---

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Prashanth-002/Resume_Builder_Using_React.git
```

### 2️⃣ Install Dependencies

```bash
npm install
```

### 3️⃣ Run the Application

```bash
npm run dev
```

---

---

## 📌 Usage

1. Enter your details in the form
2. View live preview on the screen
3. Select a resume template
4. Customize sections as needed
5. Click **Download PDF** to export your resume
